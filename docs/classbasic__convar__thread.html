Condition variables are an additon to the mini Thread classes. If you want to include them, you need to define the following in your config file with MN_THREAD_CONFIG_YES Extension of the basic thread with condition variable support

Zu beachten
Threads wait, while condition_variables signal.
Parameter
cvThe condition variable associated with the Wait.
cvlThe required condition variable lock. The Lock must be held before calling Wait.

friend class basic_condition_variable
friend

The Thread class and the basic_condition_variable class are interdependent. If we allow the basic_condition_variable class to access the internals of the basic_convar_thread class, we can reduce the public interface, which is a good thing.
